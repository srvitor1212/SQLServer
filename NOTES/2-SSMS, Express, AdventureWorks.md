
## SQL Server express

* É a edição gratuita do SQL Server.

* Tem quase todas as funcionalidades principais do SQL Server, mas com limitações (tamanho máximo do banco, consumo de memória, uso de CPU).

* Ideal para estudo, aprendizado, projetos pequenos ou aplicações locais.

* Permite rodar um banco relacional completo sem custo de licença.

<https://www.microsoft.com/pt-br/sql-server/sql-server-downloads>

## SQL Server Management Studio (SSMS)

* É a ferramenta gráfica oficial da Microsoft para gerenciar bancos SQL Server.

* Criar e alterar tabelas, bancos, procedures etc.

* Executar queries em T-SQL.

* Gerenciar logins, permissões, backups, restaurações.

* Monitorar desempenho do banco.

* Facilita o uso do SQL Server, porque sem ele você precisaria fazer tudo via linha de comando.

<https://learn.microsoft.com/pt-br/ssms/install/install>
